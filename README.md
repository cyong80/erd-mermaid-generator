# ERD Generator — Electron App

DB 스키마를 스캔해 **Mermaid ERD**를 생성하는 크로스플랫폼 데스크탑 앱.

## 기능

- MySQL / MariaDB / PostgreSQL 직접 연결 (로컬 TCP)
- 스키마 자동 스캔 (테이블, 컬럼, FK 관계)
- Mermaid `erDiagram` 실시간 렌더링 (pan / zoom)
- `.md` 파일 저장
- 클립보드 복사
- PNG / SVG 내보내기
- Mermaid 코드 직접 편집

## 시작하기

```bash
yarn install
yarn dev
```

## 빌드

```bash
# 현재 플랫폼용 패키징
yarn package

# 플랫폼별 패키징
yarn package:mac    # macOS (.dmg)
yarn package:win    # Windows (.exe)
yarn package:linux  # Linux (.AppImage)

# 출력 위치
dist/
  ERD Generator-1.0.0.dmg        # macOS
  ERD Generator Setup 1.0.0.exe  # Windows
  ERD Generator-1.0.0.AppImage   # Linux
```

## 자동 배포 (GitHub Actions)

- **태그 푸시 시**: GitHub Releases에 자동 배포 (macOS / Windows / Linux)
- **수동 실행**: Actions → Release → Run workflow → 빌드 산출물을 artifact로 다운로드

### 릴리스 생성 방법

1. `package.json`의 `version` 수정
2. 커밋 및 푸시
3. 태그 생성 및 푸시:

```bash
git tag v1.0.0
git push origin v1.0.0
```

앱 내 **자동 업데이트**(electron-updater)가 GitHub Releases를 확인합니다.

## 프로젝트 구조

```
src/
├── main/
│   └── index.js        # Electron 메인 프로세스
│                       # DB 연결 (mysql2 / pg) + IPC 핸들러
├── preload/
│   └── index.js        # Context Bridge (renderer ↔ main)
└── renderer/
    └── src/
        ├── App.jsx              # 레이아웃 & 상태
        ├── components/
        │   ├── ConnectionForm.jsx  # DB 연결 폼
        │   ├── ERDPreview.jsx      # Mermaid 렌더링 + pan/zoom
        │   ├── CodePanel.jsx       # 코드 에디터
        │   └── Toolbar.jsx         # 저장/복사/내보내기
        └── styles/
            └── index.css
```
